# Contributing to TRON-8004 SDK

Thank you for your interest in contributing to the TRON-8004 SDK! We welcome contributions from the community to help improve this project.

## How to Contribute

1.  **Fork the repository**: Click the "Fork" button on the top right corner of the repository page.
2.  **Clone your fork**:
    ```bash
    git clone https://github.com/open-aibank/tron-8004-sdk.git
    cd tron-8004-sdk
    ```
3.  **Create a branch**:
    ```bash
    git checkout -b my-new-feature
    ```
4.  **Make your changes**: Implement your feature or fix.
5.  **Run tests**: Ensure all tests pass.
    ```bash
    uv run pytest
    ```
6.  **Commit your changes**:
    ```bash
    git commit -am 'Add some feature'
    ```
7.  **Push to the branch**:
    ```bash
    git push origin my-new-feature
    ```
8.  **Submit a Pull Request**: Go to the original repository and create a Pull Request.

## Coding Standards

-   **Language**: Please use English for all code comments, documentation, and commit messages.
-   **Style**: Follow PEP 8 guidelines for Python code.
-   **Type Hinting**: Use type hints for all function signatures.

## Reporting Issues

If you find a bug or have a feature request, please open an issue on the GitHub repository. Provide as much detail as possible, including steps to reproduce the issue.
